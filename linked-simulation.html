Instructions
  1. Create a file by entering the file name, type and size (MB).
  2. Click the ALLOCATE FILE button to allocate blocks to the file on the hard disk.
    • If enough memory is available, observe how blocks are allocated and linked with pointers.
  3. Repeat steps 1 through 2 until the memory is full.
  4. To deallocate a file, enter the file name, select the file type, and then click the DELETE FILE button.
    • Observe how the linked blocks are deallocated and pointers are removed.
Note 1: To see the block details, just move your mouse over the block.
Note 2: Each block contains a pointer to the next block in the file.
Note 3: Here, each block is of 20 MB in size.
